College Event Handler - Details

GitHub Repository: https://github.com/MR-BHARATH-T/College-Event-Management

Tech Stack: HTML, CSS, JavaScript, Bootstrap 5, PHP, MySQL, XAMPP

Project Overview
The College Event Management System is a web-based application developed for digitizing and streamlining the management of college events. It facilitates the organization of inter-college activities by providing a centralized, user-friendly, and efficient solution for event creation, scheduling, registration management, and publishing of reports and articles. The application allows admins, faculty, and students to interact with event-related operations from a responsive web interface, improving coordination and reducing manual workloads. Key functionalities include modules for user management, event handling, department creation, reporting, and article management, all accessible with secure access controls.

Objectives:
1. Automate and streamline scheduling, tracking, and reporting of college events.
2. Provide responsive dashboards for admins, teachers, and students.
3. Reduce inefficiencies, errors, and administrative burdens tied to manual processes.
4. Organize, publish, and archive event-related articles, reports, and resources.

Notable Implementation Details:
  • Multi-tier architecture separating presentation, application logic, and data management.
  • Modular codebase with dedicated modules:
    • Admin controls
    • Department management
    • Event management
    • Article management
    • Reports management
    • Secure user authentication
  • PHP classes for database connection and backend operations, following robust naming and code formatting conventions.
  • Security practices:
    • SQL injection prevention
    • Password hashing
    • Secure session management
  • Responsive interfaces with user role-based access and error messaging.
  • Agile development methodology for iterative delivery and adaptation to feedback.

Hide And Seek - Details

GitHub Repository: https://github.com/MR-BHARATH-T/HIDE-AND-SEEK

Tech Stack: Java (JDK 8+), Java Swing, MySQL (via JDBC), SHA-256 for password hashing; AES for encryption; SecureRandom for password/token generation, Custom LSB algorithm for PNG/BMP/GIF, Libraries (javax.crypto, java.sql, java.awt, javax.swing, java.imageio)

Project Overview

The HIDE AND SEEK Java application is a secure desktop platform that integrates user authentication, encryption, and steganography to protect sensitive digital information. Major features include:

  • Secure login and registration with input validation, SHA-256 password hashing, and temporary password recovery.
  • Symmetric key encryption module for confidential text communication (AES-based).
  • LSB (Least Significant Bit) steganography for hiding encrypted text within images, ensuring hidden data remains protected even if the carrier is intercepted.
  • User-friendly Java Swing GUI with real-time validation, prompts, and progress indicators.
  • Password management with mandatory update on recovery, enforcing strong account security.

This layered approach combines best practices in data confidentiality, integrity, and usability for sensitive digital communications.
